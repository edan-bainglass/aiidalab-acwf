You have activated the common relaxation guide. In this guide, you will run a base ACWF relaxation workflow. Begin by opening step 1.
Step 1 provides several options for selecting the input structure. Once selected, you can visualize and edit the structure before confirming it.

Tasks

  1. Select Si2 from the Browse AiiDA database panel
  2. Confirm to continue
Note: Changing the structure after confirmation resets downstream steps.
In step 2, you configure the parameters of the workflow, including structure geometry optimization (relaxation), common workflow parameters, e.g., the calculation protocol, and settings specific to selected composite workflows.

Tasks

  1. Choose the Full geometry structure relaxation option
  2. Skip step 2.1 (composite workflow selection)
  3. Follow instructions in step 2.2
Note: The base relaxation workflow is triggered when no composite workflow is selected in step 2.1.

Tasks

  1. Keep defaults for electronic and spin settings
  2. Select the Fast protocol
  3. Confirm to continue
In step 3, you select the quantum engine used to drive SCF calculations, its codes relevant to the workflow, and the computational resources.

Tasks

  1. Select Quantum ESPRESSO as the engine
  2. Check the available local/default codes
  3. Keep nodes/cpus minimal (1 node, 1 cpu)
  4. Confirm to continue
In step 4, you provide workflow metadata (label and description) and submit the workflow.

Tasks

  1. Review/edit the label and description
  2. Click Submit to launch the workflow
In step 5, you can review workflow parameters, monitor execution status, and explore results.

Tasks

  1. Open Summary for an overview of submitted inputs
  2. Open Status to follow the workflow execution
  3. Open Results to view results once complete
The summary panel contains workflow metadata and download tools.
The status panel shows the process hierarchy and live node details.
The results panel contains per-workflow result tabs.

Tasks

  1. Open the Structure tab
  2. Compare the initial and relaxed structures
