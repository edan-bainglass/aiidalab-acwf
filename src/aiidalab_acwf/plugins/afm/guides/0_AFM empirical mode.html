You have activated the AFM empirical guide. In this guide, you will run a common workflow for atomic force microscopy (AFM) image simulation. Begin by opening step 1.

Tasks

  1. In the From SMILES tab, enter: C1=CC=CC=C1
  2. Click Generate structure
  3. Confirm to continue
Configure the common workflow and AFM plugin behavior.

Tasks

  1. Skip geometry optimization by selecting Structure as is
  2. Enable the AFM plugin in step 2.1 and enable AFM
  3. Open step 2.2 to configure calculation settings

Tasks

  1. In the Common settings panel:
    1. Select an insulator electronic type
    2. Select the fast protocol
    3. Keep remaining defaults
  2. In the AFM panel:
    1. Set mode to empirical
    2. Keep remaining defaults
  3. Confirm to continue

Tasks

  1. Select Quantum ESPRESSO as the engine
  2. Check the available local/default codes
  3. Keep nodes/cpus minimal (1 node, 1 cpu)
  4. Confirm to continue

Tasks

  1. Review/edit the label and description
  2. Click Submit to launch the workflow

Tasks

  1. Open the Results panel
  2. Switch to the AFM results panel
  3. Once the calculation is complete, click on Load results
  4. Inspect the generated plot series
